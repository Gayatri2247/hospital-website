

# 🏥 Rajput Eye Hospital Website

A responsive static website built for Rajput Eye Hospital using HTML, CSS, and modern UI layout principles. The project showcases hospital information, services, doctors, and contact features.

---

## 🌐 Live Website

👉 [https://gayatri2247.github.io/hospital-website](https://gayatri2247.github.io/hospital-website)

---

## 📌 Features

* Responsive hospital website design
* Home, About, Services, Doctors, Contact pages
* Clean UI with structured layout system
* Contact form UI (frontend only)
* Google Maps integration
* Mobile-friendly design
* Reusable CSS components

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Flexbox / Grid Layout
* Font Awesome Icons
* Google Maps Embed

## 🚀 How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/Gayatri2247/hospital-website.git
```

2. Open folder:

```bash
cd hospital-website
```

3. Open `index.html` in your browser.

---

## 🌍 Deployment (GitHub Pages)

This project is deployed using GitHub Pages:

* Source branch: `main`
* Root folder deployment

Steps:

1. Go to **Settings → Pages**
2. Select branch: `main`
3. Select folder: `/root`
4. Save
